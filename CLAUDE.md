# dashboard-cabinet

Dashboard de pilotage de portefeuille clients — à usage démonstratif.

## Stack

- HTML + CSS + JS vanilla, tout dans `index.html`
- Chart.js via CDN (pas d'autre dépendance externe)
- Aucun backend, aucun serveur — ouvrir directement dans un navigateur

## Structure

Un seul fichier : `index.html`

Deux vues navigables via la sidebar :
- **Dossier** — fiche détaillée d'un client
- **Portefeuille** — vue d'ensemble du portefeuille

## Design

- Couleur principale : `#1D9E75`
- Fond clair, typographie système sans-serif
- Style SaaS B2B épuré

## Données

Toutes les données sont fictives et codées en dur dans le HTML/JS.
Aucune API, aucune base de données.

## Contexte métier

Démonstration pour cabinets d'expertise comptable.
Cible : experts-comptables dirigeants de structures PME/TPE.
